Projects

Here are my projects throughout my academic journey.

ARNISENSE Training Dummy

A sensor-integrated Arnis training system using Raspberry Pi and wearables.

Miniature Marvel: Automated IoT Smart Mini Home

IoT-based smart home controller using ESP32 and Arduino with sensors and actuators. Also controlled by bluetooth made with MIT Inventor App.

Object-Following Robot

Built a small autonomous robot using Arduino and ultrasonic sensors that can detect and follow nearby objects. Programmed logic to control motor movement based on real-time distance readings.

0 to F Hexadecimal Counter

Designed and assembled a digital circuit using logic gates and ICs to count and display hexadecimal values (0 to F) on a 7-segment display. Practiced circuit design and debugging fundamentals.

ATX Power Supply Conversion

Modified a used ATX power supply to safely power electronics projects. Worked on identifying voltage rails, rewiring connectors, and performing output testing for prototyping use.

Game Development (Unity + C#)

Started development of a 2D mobile game using Unity engine and C#. Focused on gameplay mechanics, UI design, and explored monetization strategies and performance tuning (in progress).

Faculty Search & View System (Python Kivy + MySQL)

Created a simple GUI application using Kivy framework in Python to search and display faculty information. Integrated MySQL database for backend storage and retrieval of records.
